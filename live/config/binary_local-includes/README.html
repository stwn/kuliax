CD-ROM ini adalah LiveCD sekaligus installer/pemasang Kuliax 7.0 untuk mesin 686.
Kuliax
Kuliax adalah sebuah distribusi GNU/Linux yang dioptimasi ke penggunaan komputer desktop untuk pembelajaran, penelitian, dan pengembangan di pendidikan tinggi.

Kuliax menyertakan program-program standar komputer desktop dan juga yang spesifik ke pendidikan tinggi khususnya pada bidang-bidang yang banyak menggunakan komputer sebagai alat bantu/penunjang atau bahkan sebagai intinya.

Sebagian besar program yang disertakan di Kuliax di bawah GNU General Public License (GPL), silakan periksa masing-masing paket program untuk mengetahui lisensinya secara rinci.

Informasi tentang Kuliax 7.0 dapat dilihat pada Catatan Rilis.
Pemasangan
Kuliax dapat dipasang mandiri atau berdampingan dengan sistem operasi lainnya.

Untuk memasang Kuliax 7.0 dan mendapatkan kinerja yang cukup bagus pengguna membutuhkan spesifikasi komputer:

  • Prosesor i686 800 MHz atau lebih tinggi
  • RAM 192 MB atau lebih besar
  • Kartu grafis yang mendukung minimal resolusi 640x480
  • CDROM drive
  • Ruang cakram/disk minimal 3 GB
Langkah-langkah instalasi:
  1. Boot CD Kuliax-7.0-LiveCD
  2. Pilih opsi Install kemudian pengguna dapat memilih mode teks (Install) atau mode grafis (Graphical Install)
  3. Ikuti langkah-langkah dari pemasang secara runut dan penuh perhatian.
Catatan: jika pengguna ragu, terutama pada saat proses pembuatan partisi, diskusikan dengan kami atau teman Anda.
Isu-Isu yang Diketahui
Isu-isu yang diketahui ada di rilis Kuliax 7.0 adalah:
  • pemasang Kuliax memunculkan pesan kesalahan ketika proses format sistem berkas pada cakram keras/harddisk yang mempunyai struktur partisi "tidak normal". Solusi: pilih Ignore pada pemasang jika pesan tersebut muncul
  • fonta pada menu program berbasis Java tidak tersajikan dengan baik pada resolusi layar tertentu. Solusi: gunakan paket sun-java6 dan fontanya atau upgrade ke openjdk-6 dari rilis testing Debian.
Menambah Program
Untuk menambah program pada Kuliax 7.0 yang telah terpasang, pengguna dapat menggunakan Kuliax-7.0-RepoCD yang berisi paket-paket program pilihan.

Selain dengan RepoCD, pengguna dapat mengkonfigurasi dan menggunakan lumbung/repo paket pada peladen/server terdekat di jaringan lokal atau Internet.

Kuliax merupakan turunan Debian GNU/Linux, kompatibel dengannya, dan menggunakan lumbung paket yang sama.

Saat proses pemasangan, Kuliax telah menyertakan daftar cermin lumbung paket Debian yang dapat diaktifkan langsung melalui Synaptic atau secara manual dengan menyunting berkas /etc/apt/sources.list.
