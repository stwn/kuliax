Terima kasih telah memasang Kuliax 7.0!
Halaman ini akan memberikan sedikit informasi tentang hal-hal yang umum dilakukan di Kuliax.
Menggunakan sudo
Bagi yang terbiasa menggunakan sudo, pengguna dapat melakukannya di Kuliax yaitu dengan cara menjalankan program AddSudo pada menu K - System - AddSudo, setelah itu masukkan kata sandi root dan klik OK.

Kini pengguna dapat menggunakan perintah sudo dan kata sandi Anda sendiri untuk menjalankan program dengan hak root.

Menjalankan Layanan
Untuk menjalankan layanan yang tersedia di Kuliax, pengguna dapat menggunakan KSysV yang terdapat di menu K - System.

Misal jika ingin menjalankan peladen/server Secure Shell (SSH), pengguna dapat melakukan klik pada kolom Available Services. Sebuah jendela akan terbuka dan klik Start untuk memulai.

Seperti yang dapat dilihat pada KSysV, pengguna dapat mengubah program-program layanan mana saja yang akan dijalankan pada runlevel tertentu, dst.

Konfigurasi Jaringan
Konfigurasi jaringan di Kuliax dapat dilakukan dengan menjalankan knetworkconf pada menu K - Settings - Network Settings.

Maksimalkan jendela yang terbuka dan tekan tombol Administrator Mode di sebelah kanan OK, masukkan kata sandi, klik antarmuka jaringan yang akan dikonfigurasi, tekan tombol Configure Interface dan lakukan konfigurasi.

Jika pengguna ingin menggunakan jaringan nirkabel, cara paling mudah adalah dengan menggunakan KNetworkManager. Sebelum menjalankannya pengguna perlu memulai layanan network-manager, karena Kuliax tidak menjalankan layanan ini secara default.

Buka KSysV dan pilih network-manager pada Available Services, dan Start. Jalankan KNetworkManager dari menu K - Internet - KNetworkManager, akan muncul ikon status pada bagian bawah sebelah kanan jam digital, dan pilih access point.

Seret layanan network-manager ke Runlevel 2, pada KSysV, jika pengguna ingin menjalankannya saat boot, atau gunakan perintah update-rc.d network-manager defaults.
Menambah Program
Untuk menambah program pada Kuliax 7.0 yang telah terpasang, pengguna dapat menggunakan Kuliax-7.0-RepoCD yang berisi paket-paket program pilihan.

Caranya, jalankan program Synaptic dari menu K - System - Synaptic Package Manager, masukkan Kuliax-7.0-RepoCD, dan klik menu Edit - Add CD-ROM.

Jika ingin menggunakan lumbung/repo paket selain RepoCD, pengguna dapat melakukan konfigurasi sederhana pada daftar cermin lumbung paket Debian yang tersedia di Kuliax.

Jalankan Synaptic, pilih menu Settings - Repositories, centang pada baris lumbung paket yang terdekat dengan pengguna.

Tambahkan baris baru jika pengguna mempunyai peladen yang tidak ada di dalam daftar.
